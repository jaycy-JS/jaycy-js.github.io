01

왜 지금 W케어 시장인가

여성 헬스케어는 단발 구매가 아니라 주기·증상·관심사 기반 습관 리텐션이 작동하는 카테고리. 헬시 플레저·이너뷰티·셀프 케어로 카테고리가 확장 중이지만, 국내는 아직 통합 케어 플랫폼이 비어 있음.

  • 습관 기반 리텐션이 핵심 KPI — 단가가 아닌 재방문·재구매로 모델링되는 시장
  • 탐색 → 학습 → 구매 동선이 길어 콘텐츠·큐레이션 자산이 결정적
  • 현재 시장 점유자는 단일 카테고리 중심 — 행동 변화를 설계하는 통합 케어 자리는 비어 있음
02

왜 올영이 이 시장에서 강한가

올영은 이미 탐색형 커머스 + 라이프스타일 도메인 + 반복 방문 구조를 동시에 보유. W케어가 요구하는 자산이 이미 운영 중이라는 의미.

  • 탐색형 커머스 자산 — 매장·앱·콘텐츠가 한 동선으로 연결
  • 라이프스타일 SKU·큐레이션 노하우 — 18,000 SKU 운영 기반
  • 반복 방문 구조 — 멤버십·리뷰·매장·알림 4채널 작동
  • 콘텐츠 → 구매 연결 — 큐레이션·라이브·리뷰 자산이 이미 매출 트리거

Context

Problem

Context

숨고 신사업 스쿼드 중 견적비 외 새 수익원 검토 과제. 동시에 신규·소규모 고수는 노출 알고리즘 악순환에 갇혀 발견되지 못하는 상태. 트래픽은 충분 — 광고 재고로 전환할 수 있는지가 핵심.

Problem

soomgo 노출은 리뷰·고용수·응답률 중심 알고리즘. 리뷰 없음 ↔ 노출 안 됨 악순환에 갇힌 신규·소규모 고수가 다수.
플랫폼 입장에서도 견적비 수수료 외 새로운 수익원이 필요한 시점이었다.

Signal  ·  고수 인터뷰

신규 고수 인터뷰에서 반복된 발언:

“플랫폼에서 기다리는 것 외에 방법이 없다.”
“리뷰가 없으면 선택받기 어렵고, 선택받아야 리뷰가 생긴다.”

NPS 점수는 있었지만 “왜 낮은가”를 설명할 수 없었다.
팀마다 다른 집계 기준 · 수작업 분류 한계 · 점수 변동 원인 불명 — 세 문제가 동시에 있었다.

Context

엔카닷컴 거래·문의·진단 경험 이후 수집되는 NPS와 코멘트 데이터. 점수와 코멘트는 매주 쌓이고 있었지만, 거래·딜러·금융·보증·제품 등 여러 팀이 개선 우선순위 판단 근거로 활용하지 못하는 상태.

Problem

NPS는 운영 중이었지만 팀마다 집계 기준이 달랐고, 코멘트 분류는 수작업으로 따라잡을 수 없는 규모. 점수가 오르내려도 원인 카테고리를 가리킬 수 없어 개선 과제로 연결되지 않음.

Signal

코멘트를 직접 표본 독해하니 같은 점수 안에서도 원인이 갈렸다 — 차량 정보 · 딜러 신뢰 · 앱 UX · 거래 프로세스. 점수 변동은 결국 카테고리별 비중 변화의 결과였고, 점수가 아닌 비중을 읽어야 개선이 가능했다.

Hypothesis

분류 체계의 정교함이 자동화 품질을 결정한다. 체계를 먼저 만들고 그 위에 자동화를 얹으면, 정성 데이터도 반복 가능한 의사결정 인프라가 된다.
